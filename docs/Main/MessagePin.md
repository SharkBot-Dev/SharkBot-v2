# メッセージ固定
📍メッセージを常に最新に固定できます。<br>
以下の方法で設定できます。⤵️

# 方法１
すでに送信したメッセージを右クリックして設定する方法<br>
1. 固定したいメッセージを送信する
2. メッセージを右クリックする
3. アプリ->メッセージ固定をクリックする
4. メッセージが固定される。📍

# 方法２
スラッシュコマンドから設定する方法<br>
1. /settings lock-message trueと入力し、送信する
2. タイトルと説明を入力する
3. そのまま送信する
4. メッセージが固定される。📍

# 固定を解除する方法🗑️
削除ボタンをクリックすれば削除🗑️されます。
