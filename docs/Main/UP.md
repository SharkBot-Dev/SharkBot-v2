# UP通知
↗️ほかのBotのUPを通知することができます。<br>
以下の方法で設定できます。⤵️

# Disboardの場合
1. /bump disboard trueと入力🖊️します。
2. 送信します。⬆️
3. /bumpでバンプ！💥します。

# Dissoku (ディス速) の場合
1. /bump dissoku trueと入力🖊️します。
2. 送信します。⬆️
3. /upでUP！↗️します。

# Dicoall の場合
1. /bump dicoall trueと入力🖊️します。
2. 送信します。⬆️
3. /upでUP！↗️します。

# Dcafe の場合
1. /bump dcafe trueと入力🖊️します。
2. 送信します。⬆️
3. /upでUP！↗️します。

# 鯖チャンネル の場合
1. /bump sabachannel trueと入力🖊️します。
2. 送信します。⬆️
3. /voteで投票🗳️します。

# 無効化する場合🗑️
1. /bump サービス名 falseと入力🖊️します。
2. 送信します。⬆️
3. 無効化されます。
