# ほかのBotの招待リンクを作成
ほかのBot🤖の招待リンク🔗を作成➕できます。<br>
以下の方法で使用できます。⤵️<br>

# 使用方法
1. /bot invite botのid を入力🖊️。
2. Botの招待リンク🔗が三つ、もしくは四つ作成➕される。
3. そのBot🤖の招待リンク🔗を使って招待🚗する。

# 作成されるBotのURLの種類
1. 管理者権限☢️で招待
2. 権限を選んで🖊️招待
3. メンバーの権限✅で招待
4. 権限無し😆で招待

# 使用できない場合❌
1. そのBotが非公開の場合
2. そのBotが招待を禁止している場合
3. そのBotを使用不可にされている場合
