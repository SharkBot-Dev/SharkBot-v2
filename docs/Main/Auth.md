# 認証パネル
✅メンバーがサーバーで話し始める前に、認証をさせることができます。<br>
以下の方法で設定できます。⤵️

# ワンクリック認証パネル
ボタンを押すだけで認証が完了する認証パネルです。<br>
1. /panel auth と入力🖊️し、タイトルと説明、ロールを設定し、送信します。
2. ボタンを押して、ロールが付与されることを確認します。

# 絶対値認証パネル
画面に表示された数字の絶対値を入力することで認証が完了する認証パネルです。<br>
1. /panel abs-auth と入力🖊️し、タイトルと説明、ロールを設定し、送信します。
2. ボタンを押して、絶対値を入力し、ロールが付与されることを確認します。

# Web認証パネル
Webにアクセスすることでロールが付与される認証パネルです。<br>
特定サーバーに入っている人の認証をはじくこともできます。※1<br>
1. /panel webauth と入力🖊️し、タイトルと説明、ロールを設定し、送信します。
2. ボタンを押して、Webにアクセスし、ロールが付与されることを確認します。

# その他説明
※1 特定サーバーは、サーバーオーナーが決めることができます。
/moderation serverban、serverunbanで操作できます。
