Service Information

1. 利用規約

本サービスを利用することにより、利用者は以下の規約に同意したものとみなされます。

禁止事項:
  • スパムメール、フィッシング詐欺、マルウェア配布目的の利用。
  • 違法コンテンツ（著作権侵害、公序良俗に反するもの）への誘導。
  • 過度な自動リクエストによるサーバーへの負荷。
免責事項:

本サービスは「現状のまま」提供されます。管理者は、システムの停止やデータ消失、転送先の内容から生じるいかなる損害も責任を負いません。短縮URLの永続的な保持は保証されません。

2. プライバシーポリシー

ユーザーのプライバシー保護を重視しています。

収集する情報:
  • 入力された元のURL。
  • 解析や不正防止を目的としたアクセスログ（IPアドレス、日時等）。
情報の利用:

リダイレクト機能の提供、およびサービスの安全維持のみに使用します。法的要請がある場合を除き、第三者に共有することはありません。

3. 開発者向け情報

軽量・高速なURL短縮プラットフォームです。

技術仕様:
  • Engine: Python / Flask & SQLite3
API (Experimental):

GET /shorten?url=VALUE

JSONレスポンスにて短縮URLを返却します。外部アプリからの連携にご利用いただけます。
